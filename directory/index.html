Types of sheet music:【钢琴谱】= Piano Solo; 【钢琴伴奏谱】= Piano + Vocal(s); 【钢琴+…】= Piano + multiple instruments; 【双钢琴谱】= Piano Duet

URL Naming Convention

  1. Genre:

    A = “Anime/Media Mix”, D = “Dojin”, V = “Vocaloid”, C = “C-pop”, J = “J-pop”, M = “Movie”, T = “TV Drama”, H = “Heavy Metal Music”, L = “Latin Music”

  2. Related Media Work:

    Anime: 5-digit ID from MyAnimeList.net

    Media Mix: Same as Anime, using the first season

    Movie: 8-digit ID (after “tt”) from IMDb.com

    TV Drama (Japan): 4-digit ID from jdorama.com

    Dojin: Related media work title in 5 digits, e. g. Touhou Project -> THP

    Otherwise: [Empty]

  3. Original Artist (Performer&Composer):

    Anime/Media Mix: [Empty]

    Otherwise: If performer is the composer, just write it. Otherwise, put whichever is more famous. If the name is short, put full name (e. g. AKB48, JJ-Lin), otherwise only put the initials (Hey!Say!JUMP -> HSJ).

  4. For those with multiple musical works, use the album number / single number to distinguish. For songs in the same album/single, add the track number.

  5. Function

    Anime/Media Mix/Movie/TV Drama:

    b = “BGM”, o = “Openning (Song)”, e = “Ending (Song)”, i = “Insert Song”, c = “Character Song”

    Otherwise: [Empty]

  6. Appearing Location

    Anime/Media Mix/Movie/TV Drama:

    If it is included in OST, use “OSTxx”, where xx is the track number. When there are more than one OST, add a, b, c… to distinguish (e. g. OSTc07, the track#7 in the 3rd OST).

    If it is not included in OST, use “EPxx” for xx episode, add season “SSxx” when it is not the first season. When there are more than one from the same episode (or from the same movie at the most situation), add a, b, c… to distinguish.

    Otherwise: [Empty]

  7. Instruments/Voicings:

    P = “Sheet Music for Piano Solo”, V = “Sheet Music for Piano Accompanying with Vocal”, I = “Sheet Music for Piano Accompanying One Other Musical Instrument”, E = “Sheet Music for Ensemble (More than 2 Voices)”
